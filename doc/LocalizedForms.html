OpenType Test: Localized Forms

Look at the sea of faces below. If they all look like OT:FRA, the test is passing. If any face looks like OT:ENG, your browser has failed to select the correct letter form for a language. In that case, please hover over that face and report the appearing text as a bug to your browser vendor. For example, cy → WEL means the browser was told to render the text in Welsh, whose IETF BCP47 language tag is cy. For rendering text with localized letter forms, the browser needs to convert this to an OpenType language system tag, which would be WEL for the Welsh language. See here for background.
